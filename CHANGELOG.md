## [v2.0.0](y/compare/v1.2.0...v2.0.0) (2025-12-11)

### Добавлены функции

* Реализована возможность объединения маршрутов, трансформаций, сборщиков в Интеграции.
* Реализована возможность просмотра интергации в виде диаграммы.
* Реализованы гибкие маршруты. Маршрут может включать трансформации, получение сообщений из сторонних сервисов по http.
* Расширены возможности управления очередями. Добавлена возможность создания очередей для нескольких потребителей.
* Реализована возможность трансформации сообщений. Трансформация описывается скриптами: java script, starlark-python.
* Реализована возможность получения сообщений по http из сторонних сервисов по расписанию. Получение описывается скриптами: java script, starlark-python.
* Реализована возможность редактирования приложений, очередей, маршрутов, трансформаций, сборщиков.
* Реализована почасовая ротация журналов логирования.
* Реализовано удаление журналов логирования старше 7-ми дней.
